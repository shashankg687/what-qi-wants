💡 Motivation

Why This Problem
Matters

Explain the real-world significance of this problem. Why should students care? What impact will solving it have on the industry or society?

The knapsack problem is a well-known academic problem that has many important real-world applications. For example, perhaps a retail company needs to load containers with items so that it has the highest total value (in order to maximize profits) while respecting the containers size and weight requirements. This important optimization problem can find its roots in the knapsack problem.

Impact Statement: Even small improvements to this hard optimization problem can have large impacts on a company’s performance metrics like profits and losses.

🎯 Problem Statement

Detailed Problem
Description

Build a real-world application demo of the knapsack problem targeted towards a specific problem in a specific industry. For example, the container loading problem in the retail industry referenced in the “Motivation” section. You may choose any application in any industry, so long as the foundational, underlying problem is the knapsack problem (or a variation of it).

The problem must include, at a minimum:
• Inputs: list of items with value and weight, weight restriction
• Objective: maximize value of items selected
• Constraint: total weight of items selected less than or equal to weight restriction

Additional inputs and constraints can be added as desired for a more realistic scenario. To get everyone started, a small basic example can be found at https://github.com/dwave-examples/knapsack.
🔑 Key Challenges

List the core technical challenges students will face.

  • Research real-world requirements
    Research requirements of your chosen application for a realistic demo.
  • Map real-world requirements to model
    Map real-world constraints to a mathematical model in a Python demo.
  • Build a user-friendly interface
    Build a user-friendly demo for non-technical industry professionals.
