Detailed Problem
Description

Provide a clear and detailed description of the challenge. Be specific about what the students need to build, implement, or solve.

Twin-Field QKD is promising for long-distance secure key distribution. A critical security task is estimating parameters like the single-photon yield and error rate from experimental counts using decoy states.

Objective: Students will simulate decoy-state methods under finite-block-size assumptions and realistic imperfections (e.g., intensity fluctuations) to calculate secure key rates.

Scope & Context: You must consider finite-size bounds like Hoeffding or Chernoff bounds applied to the parameter estimation process. The simulation should yield a clear analysis of how key rates are impacted by these imperfections.
ğŸ”‘ Key Challenges

List the core technical challenges students will face.

  • Finite-size Effects Simulation
    Accurately modeling the impact of finite block sizes on key rate using rigorous statistical bounds.
  • Realistic Imperfections
    Incorporating practical non-idealities such as intensity fluctuations in the laser sources and detector inefficiencies.
  • Key Rate Optimization
    Optimizing decoy state probabilities and intensities to maximize the secure key rate.
